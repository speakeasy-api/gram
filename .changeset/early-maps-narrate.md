Add an interactive Gram customer journey map at `workflows.html` (repo root). Single-file, JSON-driven page that visualizes 11 user journeys — sign up, accept invite, invite teammate, create toolset from OpenAPI, create toolset from a function, test in playground, install MCP in Claude/Cursor, build assistant + embed Elements, connect Slack, schedule a trigger, investigate a tool call — across 22 customer-facing surfaces (routes, modals, external sites). Click a journey to highlight its path; hover a step to focus a single hop. Edit the embedded `workflow-data` script tag to add or change journeys; no build step.
