# Features

## Overview

Manage product level feature controls.

### Available Operations

* [get](#get) - getProductFeatures features
* [set](#set) - setProductFeature features

## get

Get the current state of all product feature flags.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="getProductFeatures" method="get" path="/rpc/productFeatures.get" -->
```typescript
import { Gram } from "@gram/client";

const gram = new Gram();

async function run() {
  const result = await gram.features.get();

  console.log(result);
}

run();
```

### Standalone function

The standalone function version of this method:

```typescript
import { GramCore } from "@gram/client/core.js";
import { featuresGet } from "@gram/client/funcs/featuresGet.js";

// Use `GramCore` for best tree-shaking performance.
// You can create one instance of it to use across an application.

    const { value: result } = res;

run();
```

## set

Enable or disable an organization feature flag.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="setProductFeature" method="post" path="/rpc/productFeatures.set" -->
```typescript
import { Gram } from "@gram/client";

const gram = new Gram();

async function run() {
  await gram.features.set({
    setProductFeatureRequestBody: {
      enabled: true,
      featureName: "tool_io_logs",
    },
  });


}

run();
```

### Standalone function

The standalone function version of this method:
