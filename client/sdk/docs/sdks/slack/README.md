# Slack

## Overview

Auth and interactions for the Gram Slack App.

### Available Operations

* [slackLogin](#slacklogin) - login slack
* [slackCallback](#slackcallback) - callback slack
* [deleteSlackConnection](#deleteslackconnection) - deleteSlackConnection slack
* [getSlackConnection](#getslackconnection) - getSlackConnection slack
* [updateSlackConnection](#updateslackconnection) - updateSlackConnection slack

## slackLogin
