# Hooks

## Overview

Receives hook events from coding assistants for tool usage observability.

### Available Operations

* [hooksNumberClaude](#hooksnumberclaude) - claude hooks
* [hooksNumberCodex](#hooksnumbercodex) - codex hooks
* [hooksNumberCursor](#hooksnumbercursor) - cursor hooks
* [ingest](#ingest) - ingest hooks
* [hooksNumberLogs](#hooksnumberlogs) - logs hooks
* [hooksNumberMetrics](#hooksnumbermetrics) - metrics hooks

## hooksNumberClaude

Unified endpoint for all Claude Code hook events. Handles SessionStart, PreToolUse, PostToolUse, and PostToolUseFailure.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="hooks#claude" method="post" path="/rpc/hooks.claude" -->
```typescript
import { Gram } from "@gram/client";

const gram = new Gram();

async function run() {
  const result = await gram.hooks.hooksNumberClaude({
    claudeHookPayload: {
      hookEventName: "Notification",
    },
  });

  console.log(result);
}

run();
```

### Standalone function

The standalone function version of this method:

```typescript
import { GramCore } from "@gram/client/core.js";
import { hooksHooksNumberClaude } from "@gram/client/funcs/hooksHooksNumberClaude.js";

// Use `GramCore` for best tree-shaking performance.
// You can create one instance of it to use across an application.
const gram = new GramCore();

async function run() {
  const res = await hooksHooksNumberClaude(gram, {
    claudeHookPayload: {
      hookEventName: "Notification",
    },
  });
  if (res.ok) {
    const { value: result } = res;
    console.log(result);
  } else {
    console.log("hooksHooksNumberClaude failed:", res.error);
  }
}

run();
```
