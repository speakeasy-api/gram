# Auth
(*auth*)

## Overview

Managed auth for gram producers and dashboard.

### Available Operations

* [callback](#callback) - callback auth
* [info](#info) - info auth
* [login](#login) - login auth
* [logout](#logout) - logout auth
* [register](#register) - register auth
* [switchScopes](#switchscopes) - switchScopes auth

## callback

Handles the authentication callback.

### Example Usage

```typescript
import { Gram } from "@gram/client";

const gram = new Gram();

async function run() {
  const result = await gram.auth.callback({
    code: "<value>",
  });

  console.log(result);
}

run();
```
