# Variations

## Overview

Manage variations of tools.

### Available Operations

* [createGlobal](#createglobal) - createGlobal variations
* [deleteGlobal](#deleteglobal) - deleteGlobal variations
* [listGlobal](#listglobal) - listGlobal variations
* [listGroups](#listgroups) - listGroups variations
* [upsertGlobal](#upsertglobal) - upsertGlobal variations

## createGlobal

Ensure the project-default (global) tool variation group exists, returning it. Idempotent: returns the existing group unchanged when present, otherwise creates it. Takes no parameters and only manages the single project-default group.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="createGlobalToolVariationGroup" method="post" path="/rpc/variations.createGlobal" -->
```typescript
import { Gram } from "@gram/client";

const gram = new Gram();

async function run() {
  const result = await gram.variations.createGlobal();

  console.log(result);
}

run();
```
