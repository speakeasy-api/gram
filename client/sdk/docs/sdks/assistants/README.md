# Assistants

## Overview

Manage assistants and their runtime configuration.

### Available Operations

* [create](#create) - createAssistant assistants
* [delete](#delete) - deleteAssistant assistants
* [ensureManaged](#ensuremanaged) - ensureManagedAssistant assistants
* [get](#get) - getAssistant assistants
* [getManaged](#getmanaged) - getManagedAssistant assistants
* [list](#list) - listAssistants assistants
* [sendMessage](#sendmessage) - sendMessage assistants
* [update](#update) - updateAssistant assistants

## create

Create an assistant.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="createAssistant" method="post" path="/rpc/assistants.create" -->
```typescript
import { Gram } from "@gram/client";

const gram = new Gram();

async function run() {
  const result = await gram.assistants.create({
    createAssistantForm: {
      instructions: "<value>",
      model: "Land Cruiser",
      name: "<value>",
      toolsets: [],
    },
  });

  console.log(result);
}

run();
```

## sendMessage

Send a message from the dashboard to an assistant as the calling user. Continue an existing conversation by passing its chat_id (from listChats), or omit chat_id to start a new conversation — the server mints and returns a fresh chat id. The reply is delivered asynchronously; poll the chat service (loadChat) to read it.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="sendAssistantMessage" method="post" path="/rpc/assistants.sendMessage" -->
```typescript
import { Gram } from "@gram/client";

const gram = new Gram();

async function run() {
  const result = await gram.assistants.sendMessage({
    sendMessageRequestBody: {
      assistantId: "6c62bf37-7865-4440-8031-71e7336013d5",
      message: "<value>",
    },
  });

  console.log(result);
}

run();
```
