# Logs
(*logs*)

## Overview

Call logs for a toolset.

### Available Operations

* [list](#list) - listLogs logs

## list

List call logs for a toolset.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="listToolLogs" method="get" path="/rpc/logs.list" -->
