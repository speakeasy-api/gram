# AIIntegrationConfig

Per-organization AI provider integration config. The provider API key is write-only; reads only expose whether a key is configured.

## Example Usage

```typescript
import { AIIntegrationConfig } from "@gram/client/models/components/aiintegrationconfig.js";

let value: AIIntegrationConfig = {
  enabled: true,
  hasApiKey: true,
  organizationId: "<id>",
  provider: "<value>",
};
```
