# McpEndpoint

An MCP endpoint: a url-friendly slug identifier that addresses an MCP server.

## Example Usage

```typescript
import { McpEndpoint } from "@gram/client/models/components/mcpendpoint.js";

let value: McpEndpoint = {
  createdAt: new Date("2026-11-25T22:25:42.205Z"),
  id: "54d6ecfc-1c85-405d-b299-9c70234be25e",
  mcpServerId: "6552bf74-f72a-4d02-99e6-cbc9384e4c44",
  projectId: "384cd155-2fc6-4b38-aa01-434da948c59f",
  slug: "<value>",
  updatedAt: new Date("2026-04-30T08:13:24.482Z"),
};
```
