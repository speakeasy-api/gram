# CreateMcpServerForm

Form for creating a new MCP server. Exactly one of remote_mcp_server_id, tunneled_mcp_server_id, or toolset_id must be provided.

## Example Usage

```typescript
import { CreateMcpServerForm } from "@gram/client/models/components/createmcpserverform.js";

let value: CreateMcpServerForm = {
  name: "<value>",
  visibility: "public",
};
```
