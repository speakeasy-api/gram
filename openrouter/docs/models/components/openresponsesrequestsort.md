# OpenResponsesRequestSort

The sorting strategy to use for this request, if "order" is not specified. When set, no load balancing is performed.


## Supported Types

### ProviderSort

```go
openResponsesRequestSort := components.CreateOpenResponsesRequestSortProviderSort(components.ProviderSort{/* values here */})
```

### ProviderSortConfig

```go
openResponsesRequestSort := components.CreateOpenResponsesRequestSortProviderSortConfig(components.ProviderSortConfig{/* values here */})
```

### 

```go
openResponsesRequestSort := components.CreateOpenResponsesRequestSortAny(any{/* values here */})
```
