# OpenResponsesInput

Input for a response request - can be a string or array of items


## Supported Types

### 

```go
openResponsesInput := components.CreateOpenResponsesInputStr(string{/* values here */})
```

### 

```go
openResponsesInput := components.CreateOpenResponsesInputArrayOfOpenResponsesInput1([]components.OpenResponsesInput1{/* values here */})
```
